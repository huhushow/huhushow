What is paging?

A technique that divides a large amount of data into a certain number of pieces

Why using paging?

Both human and machine would hard to handle a large amount of data

How paging works?

  1. Front-end gives to back-end their data range
  2. Back-end loads data by given range from front-end
  3. Also back-end gives summary of total data to front-end. that data used by paging function.

What do I practice?

  • Simple paging program(focus on SQL)
  • Why paging SQL getting slower?
  • think about more faster paging technique.

Simple Query

@app.route('/list/<int:page_id>', methods=['GET'])
def show_page(page_id):
    cnt = 10
    offset_num = cnt * (page_id - 1)
    db = get_db()
    cur = db.execute('select title, content from entries order by idx desc limit ? offset ?',
                    [cnt, offset_num])
    entries = cur.fetchall()
    return render_template('show_entries.html', entries=entries)
